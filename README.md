# 🏖️ Chennai Tourism Guide

Welcome to the **Chennai Tourism Guide** - Your ultimate companion to explore the vibrant city of Chennai! Discover amazing tourist destinations, must-try food spots, and exciting sightseeing opportunities.

## 🌟 Features

- 📍 **Must-Visit Places** - Iconic landmarks and tourist attractions
- 🍽️ **Food Recommendations** - Best restaurants and street food spots
- 🎭 **Sightseeing Spots** - Beautiful locations for photography and exploration
- 📱 **Fully Responsive** - Beautiful design on all devices
- ✨ **Smooth Animations** - Modern, engaging user experience
- 🔍 **Easy Navigation** - Intuitive and user-friendly interface

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/tarun7explorer/Chennai-Tourism.git

# Navigate to project directory
cd Chennai-Tourism

# Install dependencies
npm install

# Start the development server
npm start
```

The site will open at `http://localhost:3000`
